trait EventLike[+T, S <: Struct, SL[+X, Z <: Struct] <: SignalLike[X, Z, SL, EV], EV[+X, Z <: Struct] <: EventLike[X, Z, SL, EV]] extends AnyRef

Base signal interface for all signal implementations. Please note that any event implementation should have the EV type parameter set to itself and be paired with exactly one signal implementation it is compatible with by setting the SL type parameter. This relationship needs to be symmetrical.

T

Type returned when the event fires

S

Struct type used for the propagation of the event

SL

Signal type supported as parameter and used as return type for event methods

EV

Event type supported as parameter and used as return type for event methods
