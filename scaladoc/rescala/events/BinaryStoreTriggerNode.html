BinaryStoreTriggerNode

abstract class BinaryStoreTriggerNode[StoreType, TriggeredType, T1, T2] extends EventNode[TriggeredType] with DepHolder with Dependent

takes two events and combines their values with the storeA and storeB methods can filter and map the combined result with the trigger method the current state is reset after every turn
